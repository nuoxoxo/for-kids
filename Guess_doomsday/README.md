# :sunflower: Guess the Doomsday :sunflower: 

Given a `year` (YYYY) \
Guess the Doomsday of that Year
- What is a doomsday?
  - cf. [Here](https://en.wikipedia.org/wiki/Doomsday_rule)

```rust
➜ home $ cd Guess_doomsday
➜ Guess_doomsday $ python3 guess_doomsday.py 
```
